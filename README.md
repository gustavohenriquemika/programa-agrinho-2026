# EcoAgro Sim - Jogo Educativo (Concurso Agrinho)

Este projeto foi desenvolvido para o Concurso Agrinho na categoria Programação/Alura.

## 🌾 Tema do Projeto
"Agro forte, futuro sustentável: equilíbrio entre produção e meio ambiente".

## 🎮 Como Jogar
1. Abra o jogo e pressione **ENTER** para iniciar.
2. Leia os dilemas rurais enfrentados pelo produtor em cada rodada.
3. Use a tecla **A** para tomar decisões focadas em alta produção imediata ou a tecla **B** para investir em práticas sustentáveis.
4. O objetivo é chegar ao final sem falir o caixa da fazenda e sem deixar o indicador ecológico cair abaixo de 40%.
